## ğŸ”§ Quick Customizations

### Change Text
1. Open `index.html` in any text editor (Notepad, TextEdit, VS Code)
2. Find the text you want to change
3. Edit it
4. Save the file
5. Refresh your browser to see changes

### Change Colors
1. Open `styles.css` in a text editor
2. Find the `:root` section at the top
3. Change the color codes:
   ```css
   --gold: #c49356;
   --purple: #46215f;
   --red: #8b1d1d;
   ```
4. Save and refresh

### Replace Images
1. Add your new image to the `images/` folder
2. Name it clearly (like `new-logo.png`)
3. Open `index.html`
4. Find: `<img src="images/old-image.png">`
5. Change to: `<img src="images/new-logo.png">`
6. Save and refresh
